  This example is based on the GD32E230C-EVAL board, it provides a description 
of how to use Tamper key to control the LED2. The example uses polling mode. 
  Press the key, LED2 will be on. Press the Key again, LED2 will be off.

  On the GD32E230C-EVAL board,the Tamper key connected to PC13 and the LED2 connected 
to PA11.
